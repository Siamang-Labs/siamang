## 1. What Siamang Cloud is

Siamang Cloud is a hosted platform for designing, deploying and analysing surveys
defined as code with the `siamang` engine. It lets you:

- write questionnaires, analysis scripts and project configuration in a Git
  repository;
- validate and preview them, deploy live surveys, and collect responses;
- run analysis in a sandbox and export results;
- work in organizations with teammates, and move data in and out via connectors.
